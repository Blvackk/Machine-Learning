It is divided into 2 types -->
1. Standard Scalar | z-score | range -3 to 3
2. Normalization | min max scaler | range 0 to 1


Normalization is one of feature scaling technique which implement in ML model to get Highest accuracy.  It normalize value & shift it between 0 to 1
Hence, Normalization also called as ( normalizer OR  min max scaler )


Standarization is another feature scaling technique in which 
  standard scaler because apply z-score
  In ML , every value are shifted between -3 to 3 

For every model you can apply both ,
but 1st priority is always given to [ standardization ] bcoz -> it follows GAUSIAN DISTRIBUTION
then Normalization -> does not follow GAUSIAN DISTRIBUTION
